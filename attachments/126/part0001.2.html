> Comment (by Anthony Chan):
>
>  Look like you have 2 different versions of gnu compiler
> suite installed
>  on your linux box.  The gcc/g++ are 4.1 and gfortran
> is 4.4.  I guess
>  these
>  2 different version of gnu compilers are not working
> nicely together.
>  I would think you should try using both C and fortran
> compilers from
>  the same version of GNU suite in configuring mpich2
> through setting of
>  CC, CXX, F77 and F90, i.e.
>
>  ..../configure CC=<path2gnu>/bin/gcc
> CXX=<path2gnu>/bin/g++ \
>           
> F77=<path2gnu>/bin/gfortran F90=<path2gnu>/bin/gfortran \
>            --
> prefix=<....>
>  Be sure to have the shared libraries of the compiler found
> by the system
>  if your system have multiple versions of the same
> compiler, i.e. use
>  LD_LIBRARY_PATH.
>
