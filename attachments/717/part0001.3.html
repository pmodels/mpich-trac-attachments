 If you want to use "-user 1" option with mpiexec when running your jobs (mpiexec -user 1 -n 1 cpi.exe), register your username/password using the "-user 1" option (mpiexec -register -user 1).
 I would recommend the following,
 
# Make sure you have the same user (reza) on both the machines and that they have the same password (Assuming reza is not a domain user).
# Register your username/password using "-register" option of mpiexec WITHOUT the "-user" option (mpiexec -register).
# Run your MPI jobs WITHOUT the "-user" option (mpiexec -n 2 cpi.exe)
# Use wmpiconfig if needed.
 
 Let us know if it works for you.
