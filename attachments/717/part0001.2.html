 The problem seems to be with the username/password registered with MPICH2 (incorrect username or password). Note that if the user is a domain user the domain name has to be provided when registering your username/password.
 You can validate the username/password registered with MPICH2 using the "-validate" (mpiexec -validate) option of mpiexec. Try re-registering/validating your username/password from command line (Type "mpiexec -regsiter" & "mpiexec -validate" from the command prompt) and let us know the results.
