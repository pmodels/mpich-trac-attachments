Hi,
  There could be two reasons you are getting the error message,

1) You don't have the Visual studio runtime libs installed in your machine. Please install the Visual Studio SP1 x64 redistributable package from microsoft (http://www.microsoft.com/downloads/details.aspx?familyid=EB4EBE2D-33C0-4A47-9DD4-B9A6D7BD44DA&displaylang=en).

2) If you still have problems try installing the MPICH2 process manager, smpd, from an admin command prompt (Right-click on shortcut to command prompt & Select "Run as administrator". You can install smpd by typing, smpd -install, at the admin command prompt. Smpd.exe should be available in MPICH2\bin).

 Let us know if it works for you.
