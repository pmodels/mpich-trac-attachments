Hi,
  You will need Intel fortran compiler which can create 64-bit executables (The compiler itself does not have to be 64-bit) if you need to compile fortran MPI programs using MPICH2. You need to install Visual Studio (The compiler allows you to create 32-bit & 64-bit code) compiler if you want to compile C MPI programs using MPICH2.

(PS: Intel Visual Fortran compiler also comes with a Visual Studio plugin. After installing the plugin you can compile your fortran programs from within visual studio IDE.)
