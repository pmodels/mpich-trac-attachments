 Hi,
  You can run MPI programs with the processes spanning windows and linux machines if the machines involved have the same data model (eg: You can run your MPI program on x86 linux machines and x86 windows machines. You cannot run your MPI program on x86 linux machines and x86-64 windows machines.).
  If you want to run your MPI program across linux and windows machines try the following steps,

# Install MPICH2 on windows (MPICH2 on windows has SMPD as the process manager. MPD is not available on windows.)
# configure MPICH2 on unix to use SMPD as the process manager (--with-pm=smpd), make/install MPICH2.
# Run your MPI program using the "-machinefile" option of mpiexec making sure that the path of the executables is specified correctly using the "-path" option of mpiexec.

 mpiexec -n 2 -path "/home/myhome;c:\temp" -machinefile mf.txt cpi.exe

  Let us know if you need further help.
