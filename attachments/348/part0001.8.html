 Sorry that you had to find the binary incompatibility issue during

 linking,

 mpich2 actually has a test for the binary compatibility between C

 and fortran compilers but the test isn't included in 1.0.8

 (but 1.1 alpha releases).

 Anyway, you could set all compilers to use same ABI by setting CFLAGS,

 CXXFLAGS, FFLAGS, and F90FLAGS.  Since your configure output shows that

 configure chose gcc as CC, g++ as CXX, g77 as F77 and SGI's f90 as F90,

 the *FLAGS that you set should correspond to the same ABI format.

 It may be easier if you use all SGI compilers, i.e. set CC=cc, F77=f77 and

 F90=f90... so all understand -n32 or -64 ABI flags (I haven't had access

 to

 any SGI box for a long time, so info provided above may not be up-to-

 date.)
