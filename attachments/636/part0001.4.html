Hi,

As I said before: "Unfortunately .NET (/clr) and multithreaded static lib (/MT) are incompatible, you cannot even compile the project."

Visual Studio says: "Command line error D8016: '/MT' and '/clr' command-line options are incompatible". So the only way to compile a .NET project is to use the _dynamic_ runtime libraries. MPI works just fine with the static runtime libraries, but my issues were with MPI in combination with .NET.
