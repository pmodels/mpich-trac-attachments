Hi,
  Did you get an error message while installing MPICH2 on your machine ?
  Please check whether you have the process manager, smpd, running on your system using the command, "smpd -status". If your installation was successful the process manager, smpd, should be installed as a service on your system.
  Windows Vista only allows administrators to install a service on a system. So if "smpd -status" returns an error this is most likely your problem. You can install the process manager manually by opening a command prompt in Vista as an administrator (Click on "Start"-->"Programs" --> "Accessories", Right-Click on "command prompt" and choose "Run as administrator" ). From within the command prompt type "smpd -install" to install the process manager.
  Let us know if it works for you.
