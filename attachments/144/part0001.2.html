Comment (by Anthony Chan):

 Your configure.log has these lines:

 ....
 configure: configuring in src/graphics
 ....
 checking for X... no
 1064    configure: WARNING:  Could not find appropriate X windows
 libraries.  MPE routines that make use of X11 graphics will NOT be built
 unless you put X windows libraries in your path and reconfigure!
 ....

 Do you have X11 header files and X11 libraries installed on your linux box
 ?
 Your system probably does not have X11 development package which contains
 the
 needed header files.  If they are installed in non-standard location, you
 can
 use configure command options to point configure to them

  --x-includes=DIR    X include files are in DIR
  --x-libraries=DIR   X library files are in DIR

 A.Chan
